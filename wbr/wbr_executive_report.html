Executive Dashboard Framework

🎯 Recommended Weekly Dashboard Structure

📈 Performance Trends (WoW/MoM)
  • Primary KPIs: Orders, Bookings, CAC
  • Growth Metrics: % change vs previous period
  • Efficiency: Spend per order, Conversion rates
  • Format: Cards with trend indicators (↑↓)
🌍 Regional Performance
  • Growth Leaders: Top 3 regions by growth %
  • Volume Leaders: Top regions by absolute orders
  • Opportunities: Declining regions needing attention
  • Format: Table with sparklines for trends
🔮 CAC Forecasting
  • 4-Week Projection: Expected CAC with spend scaling
  • Scenario Planning: 10%, 20%, 50% spend increases
  • Elasticity Model: Diminishing returns curve
  • Format: Charts with confidence bands
Implementation Recommendations

Daily sales CSV + Regional sales CSV automated imports

Daily data refresh, weekly executive summary

Alert Thresholds:

CAC >10% WoW increase, Orders <-15% WoW decline
